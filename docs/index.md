# GISMentors: QField pro každého

![](images/qfield_for_qgis_logo.png){align=left}

[QField](https://qfield.org/) je open source *geografický informační
systém* ([GIS]({{wikipedia}}/GIS)) publikovaný pod všeobecnou licencí GNU GPL.
Projekt vznikl na základě potřeby umožnit mapování v terénu
s napojením na QGIS. Díky tomu je dnes možné mapovat v terénu s
využitím mobilního telefonu a data synchronizovat s dekstop aplikací.
